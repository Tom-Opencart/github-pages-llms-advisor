LLMS Setup Advisor

Вставьте ссылку на сайт и получите стартовый набор инструкций именно под модуль LLMS.txt Generator для OpenCart.

Что это делает: аккуратно собирает sitemap, сервисные страницы, новости, обзоры, приоритетные ссылки и готовые тексты для AI-блока.
Чего не обещает: это помощник по настройке, а не гарантия идеальной индексации и не волшебный генератор результата.

1. Анализ сайта

Инструмент работает полностью на клиенте и подходит для размещения на GitHub Pages.

Статус: Ожидает URL сайта.

Как это работает на GitHub Pages

  • Сначала инструмент пробует читать сайт напрямую через браузер.
  • Если сайт блокирует CORS, включается безопасный fallback через публичный текстовый прокси.
  • Если и это не помогает, вы всё равно получите полуавтоматический шаблон на основе URL и ваших подсказок.
